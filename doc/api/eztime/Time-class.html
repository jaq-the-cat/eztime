Time

Time class Null safety

Represents a Time in hours, minutes and seconds.

Automatically wraps values, so both Time(0, 60, 60) and Time(24, 60, 60) will return 01:01:00.

This class supports addition, subtraction and conversion to and from strings and milliseconds, aswell as addition with days.
